## 面试题:事件循环1
```javascript
Promise.resolve()
  .then(() => {
    console.log(0)
    return Promise.resolve(4)
  })
  .then((res) => {
    console.log(res)
  })

Promise.resolve()
  .then(() => {
    console.log(1)
  })
  .then(() => {
    console.log(2)
  })
  .then(() => {
    console.log(3)
  })
  .then(() => {
    console.log(5)
  })
  .then(() => {
    console.log(6)
  })
```
>输出结果是: 0 1 2 3 4 5 6
>
核心关键:
1.`Peomise.resolve().then()`进入微任务队列
2.`return Promise.resolve(4)` 的额外微任务开销

PromiseResolveThenableJob 是 Promise 规范（ECMAScript）中定义的一种微任务（Microtask），专门用于处理一种特殊情况：
>返回 Promise.resolve(4)
    ↓
① 调度 PromiseResolveThenableJob（tick +1）
    ↓
② 该 Job 调用 p4.then(resolve)，因 p4 已 resolved，再调度 resolve（tick +2）
    ↓
③ resolve 执行，链上的下一个 .then(cb_A2) 才进队

`return Promise.resolve(4)`相较于`return 4`多消耗了两个额外的微任务tick
